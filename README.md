### This is a full stack MERN application called "instaJam" (JAM is my nickname.) It is a simple social media app that allows users to create an account or login to existing accounts and create, edit, modify or delete and post they have created. Users are able to upload photos and share cool things they have done.

#### Front end: React // Redux // Redux Thunk // HTML // CSS // Material-UI // Axios 

#### Back end: Node.js // MongoDB // Express // Mongoose // Nodemon
